</span><span class="approved-insertion" data-user="34" data-username="SteulK" data-date="26319530">Hier

Einsatzstellen bei einem Unwetter nicht angefahren werden können oder
Liegenschaften überflutet werden. Dies kann die Hilfeleistung verzögern
und die Versorgungssicherheit der Bevölkerung
gefährden.
